关于hexo博客的便捷式改进

Posted on 2021-02-07 | In tools
Words count in article: 185 | Reading time ≈ 1

提升hexo在本地计算机上使用的便携性。

hexo-admin安装

  1. 安装hexo-admin插件；
  2. config.yml中设置admin登录的用户名，密码，密文
  3. 设置deployCommand命令，设置后即可通过web端deploy

本地启动node服务

  1. 创建vbs脚本

    set ws=WScript.CreateObject(“WScript.Shell”)

    ws.Run “D:\Blog\hexo\hexo-server.bat /start”,0

  2. 创建bat脚本

    D:

    cd D:/Blog//hexo

    hexo s -d

  3. vbs放在目录

    C:\Users\$username$\AppData\Roaming\Microsoft\Windows\Start Menu\Programs\Startup

  4. bat放在目录

    D:\Blog\hexo

Bug

如果本地设置了ssh私钥的密码，则上述配置deploy时node会意外退出；但是如果通过gitbash打开server端后deploy则不会出现该问题。
我怀疑是vbs脚本这里中WScript有点问题。
